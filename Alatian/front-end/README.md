# الاتيان - متجر قطع الكمبيوتر

موقع إلكتروني متكامل لبيع قطع الكمبيوتر والإكسسوارات مبني باستخدام Next.js 15.

## المميزات

- ✅ Next.js 15 مع App Router و TypeScript
- ✅ Tailwind CSS للتصميم
- ✅ دعم RTL (من اليمين لليسار) للغة العربية
- ✅ تصميم متجاوب (Mobile First)
- ✅ صفحات متعددة: الرئيسية، الفئات، الماركات، البحث، السلة، الحساب
- ✅ مكونات قابلة لإعادة الاستخدام
- ✅ بيانات تجريبية جاهزة

## التقنيات المستخدمة

- **Next.js 15** - إطار عمل React
- **TypeScript** - للكتابة الآمنة
- **Tailwind CSS** - للتصميم
- **Lucide React** - للأيقونات

## التثبيت والتشغيل

1. تثبيت المتطلبات:
```bash
npm install
```

2. تشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

3. فتح المتصفح على:
```
http://localhost:3000
```

## بناء المشروع للإنتاج

```bash
npm run build
npm start
```

## هيكل المشروع

```
├── app/                    # صفحات Next.js
│   ├── page.tsx           # الصفحة الرئيسية
│   ├── categories/        # صفحة الفئات
│   ├── brands/            # صفحة الماركات
│   ├── search/            # صفحة البحث
│   ├── cart/              # صفحة السلة
│   └── account/           # صفحة الحساب
├── components/            # المكونات القابلة لإعادة الاستخدام
│   ├── Navbar.tsx
│   ├── Footer.tsx
│   ├── ProductCard.tsx
│   ├── BrandCard.tsx
│   ├── CategoryStrip.tsx
│   └── EmptyState.tsx
├── data/                  # البيانات التجريبية
│   ├── products.json
│   ├── categories.json
│   └── brands.json
└── lib/                   # الأدوات المساعدة
    └── utils.ts
```

## الصفحات المتوفرة

- `/` - الصفحة الرئيسية مع العروض والمنتجات
- `/categories` - عرض جميع الفئات والمنتجات مع البحث والفلترة
- `/brands` - عرض جميع الماركات
- `/search` - البحث في المنتجات
- `/cart` - سلة التسوق
- `/account` - تسجيل الدخول / إنشاء حساب
